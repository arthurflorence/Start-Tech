>> COMEX <<

1. Visão do produto

O projeto Comex é um sistema de e-commerce marketplace. Seu objetivo é permitir que a sua base de cliente tenha acesso a vários vendedores e, assim, consolidar-se como o shopping eletronico mais popular da internet.

2. Entidades importantes

  • Produto
  • Categoria
  • Cliente
  • Pedido
  • ItemDePedido

3. Requisitos não funcionais

  • Manutenibilidade

4. Requisitos funcionais

Os requisitos funcionais são classificados em três categorias no Comec:

Essncial: implementação obrigatótia para o sistema atingir seu obetivo principal;

Importante:acrescentam muito valor ao negócio;

Desejável:incrementam usablilidades e funcionlidades.
